## Installation (5 minutes)

### Step 1: Create Icons

Before loading the extension, you need to create icon files:

1. Create a folder named `icons` inside the `SafeBrowsing-Extension` folder
2. You need three icon files (shield or lock design):

   * `icon16.png` (16x16 pixels)
   * `icon48.png` (48x48 pixels)
   * `icon128.png` (128x128 pixels)

**Quick Option:** Use an online icon generator:

* Visit: https://favicon.io/favicon-generator/
* Create a shield emoji or text-based icon
* Download and rename the files
* Place them in the `icons` folder

**Alternative:** Use simple colored squares temporarily:

* Any image editor can create these
* Just make solid colored squares in the right sizes

### Step 2: Load Extension in Browser

**For Chrome:**

1. Open Chrome
2. Go to `chrome://extensions/`
3. Enable "Developer mode" (toggle top-right)
4. Click "Load unpacked"
5. Select the `SafeBrowsing-Extension` folder
6. Done! ‚úÖ

**For Edge:**

1. Open Edge
2. Go to `edge://extensions/`

6. Done! ‚úÖ

**For Brave:**

## Troubleshooting

### Icons Not Showing

* Make sure you created the `icons` folder
* Check that icon files are named correctly
* Try reloading the extension

### Sites Not Blocking

* Verify the domain is in the blocked list
* Check category filters are enabled
* Try blocking the exact domain you're visiting

### PIN Not Working

* Make sure you're entering the correct PIN
* If forgotten, remove and reinstall extension (clears all data)

### Stats Not Updating

* Refresh the popup/dashboard
* Check that you're browsing regular websites (not chrome:// pages)
* Wait a minute and check again
